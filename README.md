# Blue Yellow Opera Theme

## Introduction :

This is a small Opera GX interface mod that utilises a blue yellow colour pallete because that is a pallete that can be seen across most spectrums of colourblindness,
Additionally this theme uses the Roboto font as it is more legible than Opera GX's standard Pixel font,
Finally, this theme uses custom cursors created by hm-a-technologies as the default cursors can be intrusive.

This theme comes in both light and dark modes.

## Installation : 

1. Go into Opera GX's Extensions Page 
    ( You can find it in the sidebar or by going into settings at switching to the extensions at the top. )
2. Turn on Developer Mode
3. Extract the theme from the zip
4. Click load unpacked and select the Blue Yellow Opera Theme folder ( with the manifest.json in it )

You're done. 

## Alternates : 

This theme mod comes with two different cursor sets and 3 different fonts, they can be selected and swapped by specifically chosing them in the interface section of the mods tab.
